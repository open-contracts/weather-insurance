# Weather Insurance
This contract allows anyone to provide a trustless rainfall insurance for anyone else on earth.

Would love to accept PRs adding e.g. support for [fire insurance using this real-time NASA data feed](https://worldview.earthdata.nasa.gov/?v=-227.166567355547,-113.56806782384763,177.91460589598017,121.72908281176169&l=VIIRS_SNPP_Thermal_Anomalies_375m_Day,VIIRS_SNPP_Thermal_Anomalies_375m_Night,MODIS_Combined_Thermal_Anomalies_All,Reference_Labels_15m(hidden),Reference_Features_15m(hidden),Coastlines_15m,VIIRS_SNPP_CorrectedReflectance_TrueColor(hidden),MODIS_Aqua_CorrectedReflectance_TrueColor(hidden),MODIS_Terra_CorrectedReflectance_TrueColor&lg=false&sh=VIIRS_SNPP_Thermal_Anomalies_375m_Day,C1392010612-LPDAAC_ECS&t=2021-12-29-T16%3A11%3A11Z)!
